# FLYRANK AI Capstone

An AI-assisted frontend application developed as part of the FLYRANK AI Frontend Internship.

## Project Status

The project is currently in the setup phase. Application source code and tooling are not yet in the repository.

## Planned Stack

- React
- TypeScript
- Vite
- Node.js
- CSS
- Git and GitHub
- Cursor for AI-assisted development

## Goals

- Build a responsive and accessible frontend application
- Apply reusable component-based architecture
- Use AI tools responsibly during development
- Maintain a clean and understandable Git history

## Getting Started

1. Clone this repository to your local machine.
2. Review [CLAUDE.md](./CLAUDE.md) for development conventions, Git guidelines, and AI-assisted workflow expectations.

**Prerequisites:** Node.js (LTS recommended) — required once the Vite project is initialized.

Install and run instructions will be added after the application scaffold is in place.
